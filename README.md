# 📖 [Enumerable] <a name="about-project"></a>

**[Enumerable]** is a project where we learnt how to use a module inside a class. For this we created a class "MyList" and a module "MyEnumerable". The module "MyEnumerable" will implement a subset of the functionality of Enumerable.
